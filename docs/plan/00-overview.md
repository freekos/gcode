# gcode — карта фаз

> Как читать: одна фаза = один документ ≤5 минут чтения. Я не начинаю фазу, пока ты её не прочитал и не сказал «го». Статусы: ⏳ не начата · 📖 на твоём чтении · 🔨 в работе · ✅ готова.

| # | Фаза | Что получишь в конце | Статус |
|---|------|----------------------|--------|
| 0 | [Скелет](phase-00-skeleton.md) | репо, CI, `gcode --version` | ✅ |
| 1 | [Ядро: домен + состояние + worktrees](phase-01-core.md) | из CLI создаёшь мультирепо-задачу — папка со всеми worktree готова, всё покрыто тестами | ✅ |
| 2 | [Агенты и threads](phase-02-agents.md) | задача с живым агентом (Claude Code), стриминг, resume, git запрещён агенту | 🔨 |
| 3 | Минимальный UI | доска проектов/задач, создание задачи, чат-тред — начинаешь жить в приложении | ⏳ |

| 5 | Git-ops | по готовой задаче: «Создать PR/MR» или «смёржить локально в ветку», с подтверждениями | ⏳ |

Правила этого формата: фаза >5 минут чтения → дробится; шаги внутри фазы — чекбоксы, обновляются по ходу; вопросы к тебе — всегда последняя секция.
