## Журнал правок по живому смоуку и дизайн-ревью Газика (обновляется)
- Промпт создания = первое сообщение агенту автоматически (был пустой тред).
- Provisioning ускорен: репо параллельно, fetch/deps в фон — 4 репо за 0.06s (было десятки секунд); тайминги в journal.
- Тул-события: подряд идущие сворачиваются в блок «⚙ N шагов · Bash ×k» с деталями (`Bash · pnpm test`); палитра — уникальные ключи (краш на дублях названий).
- ИИ-нейминг: ветка по git-конвенции (англ.), транслит только фолбэк; slug русских промптов.
- ZCode-паритет: нейтральный графит (пипеткой #161616/#2a2a2d), стеклянные white-кромки + инсет-хайлайты, vibrancy-сайдбар, рамка окна 8px (drag только сверху), traffic lights (19,22) в 44px шапке, overlay-тайтлбар, hub-композер по центру с чипом проекта, плоские хоткеи-текст, шапка «Проекты» (+ проект / вид-сортировка / архив-тогл), hover-кнопки строки проекта (Finder/свернуть/новая задача), лёгкие строки задач со временем, аккордеоны, ресайз сайдбара, per-panel скролл, модалка по центру (fix Tailwind preflight), error-overlay + devtools по требованию (авто-открытие убрано), низ сайдбара: аватар+имя | телефон (remote, фаза 10) | настройки.
- Hover строки проекта — на весь ряд (кнопки не «выключают» подсветку); тултипы = глобальные ZCode-пилюли через data-tip (нативные title убраны) + aria-label; композер двухслойный 1в1 (чип проекта над внутренней панелью, ряд [+][права оранжевым][движок][серая ↑]).
- Тултипы: глобальный fixed-менеджер с прижимом к краям (были ::after — клипались панелью); рамка окна 10/14px; traffic lights (26,26).
- Update-пилюля в шапке сайдбара: реальный чек GitHub Releases при старте — зелёная Update с hover-поповером release notes (клик = открыть релиз) либо тихая версия; установка — фаза 9.
- Layout по ZCode: сайдбар прямо на vibrancy-стекле окна (без фона/рамки), центр+контекст-панель = плавающая непрозрачная карточка (r16, стеклянный ободок, тень) с отступами — фон рабочего стола просвечивает по-разному, как в референсе.
- Ядро попутно: PROGRESS.md в provisioning + правило агенту, set_task_title, task_context (per-repo факты + прогресс), RateLimit-событие (окно+resetsAt).
